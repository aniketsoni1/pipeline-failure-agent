# Change Log

All notable changes to the **Pipeline Failure Agent** extension are documented here.

## [0.1.1]

- New Marketplace listing: custom icon, richer overview, feature list, Getting Started guide, and command/settings reference.
- Added `repository`, `homepage`, and `bugs` links so the Marketplace Resources panel is populated.
- No functional changes to the extension itself.

## [0.1.0]

- Initial release.
- Activity Bar panel with **Connections** and **Recent Failures** tree views.
- Investigate the active `.log` file, or a platform run by id.
- Ranked root-cause hypotheses with transparent confidence and labeled evidence, rendered in a webview.
- Export a Markdown incident report.
- Approval-gated Jira issue creation.
- Secret/PII redaction before any analysis; optional AI off by default.
- Shares the same core engine as the `pipeline-agent` CLI.
